# Setup Guide

## Quick Start

1. Clone repository
```bash
git clone https://github.com/yourusername/research-company-data-engineering.git
cd research-company-data-engineering
```

2. Install dependencies
```bash
pip install -r requirements.txt
```

3. Run pipeline
```bash
python src/solution_realistic.py
```

## GitHub Setup

1. Create new repository on GitHub (do NOT initialize with README)
2. Run these commands:

```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/YOUR_USERNAME/research-company-data-engineering.git
git branch -M main
git push -u origin main
```

3. Update README.md with your actual name, GitHub username, and LinkedIn
